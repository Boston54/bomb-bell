This website works with a mod that shares bomb information sourced from Wynntils. Because of this, the mod relies on Bomb Bell notifications, which means the Champion rank is needed to send all required information to the app. Players without Champion can still report bombs on their current world.
No user data is stored, and the mod only sends the information needed to make the site work.
The mod can be downloaded below. Note that it requires Wynntils to also be installed in order to function.
By using this mod, you agree to not AFK with the mod running, as this would break Wynncraft's terms of service.
